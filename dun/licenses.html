PVDungeons

  1. The game concept is inspired by Angband (video game) , Sil and Sil-q .
  2. The data structures and simulation algorithm are available as open source in the sil-q iOS Framework . (Forked from Version 1.5 of sil-quark's Sil-Q ). The iOS user interface code is built on the YUS Game Engine, which is designed specifically for iOS. As a result, an Android version is not available. Since the YUS Game Engine is closed-source, publishing the iOS UI code would be of little use to those without access to the engine.
