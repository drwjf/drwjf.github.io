Mapcasts iOS App

Mapcasts (full title: Mapcasts - AM/FM Radio Stations) is a privacy-focused internet radio app for iOS (iPhone, iPad, Mac, and Apple Vision). Formerly known as RGRadio, it provides access to over 50,000 AM/FM radio stations worldwide.

Core Features

  • Browse and listen to global radio stations via an interactive map or list views.
  • Features include favorites, random station selection, station search/filtering, “Hottest”/“Highest” lists, and CarPlay support.
  • Displays current song titles and album art when available.
  • Additional tools: recording streams for later playback (Premium), blocking stations, timezone-aware display, and more.
  • Strong emphasis on privacy: No ads, no tracking, no data collection, no registration, and no push notifications.

Key Details

  • Category: Music (Age rating: 4+).
  • Free with in-app purchases for Mapcasts Premium (one-time or subscription options) and tips. A separate “No Advertisement” version is also available.
  • App size: ~43.8 MB.
  • Requires iOS 15.0+ (also supports macOS and visionOS).
  • Supports English + 32 other languages.
  • Actively updated with new stations, UI improvements, themes, CarPlay enhancements, iCloud sync for favorites, and stability fixes (latest updates in 2025–2026).

Overall

Mapcasts is a simple, no-frills, map-centric internet radio player ideal for global listening.
