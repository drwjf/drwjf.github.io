Building the Encyclopedia System for BerryBelts

One of the goals of BerryBelts is to make progression feel discoverable instead of overwhelming. As the factory grows and more systems unlock, players need a place to revisit mechanics, understand buildings, and learn advanced production concepts without leaving the game.

To solve this, BerryBelts includes a built-in Encyclopedia.

⸻

Welcome to BerryBelts

Harvest, Transform, Automate

BerryBelts is a factory-building and automation game where players collect fruit items and flower pigments, process them into increasingly sophisticated creations, and deliver them to the Hub to unlock new technologies.

Built on an endless world with no enemies, deadlines, or experimentation costs, BerryBelts encourages players to optimize entirely at their own pace.

⸻

The Core Loop

At the center of every world is the Hub.

The Hub continuously requests specific items for delivery.

Early goals focus on simple items and basic pigments, but later objectives introduce:

  • Multi-layer item construction
  • Advanced coloring workflows
  • Automation systems
  • Intelligent routing
  • Signal processing

Each completed delivery unlocks new buildings and expands factory capabilities.

⸻

Harvest, Process, Create

Production starts with harvesting:

  • Fruit items from resource fields
  • Flower pigments from pigment fields

These resources can then be transformed using processing buildings:

  • Cut items into smaller parts
  • Rotate items into new orientations
  • Apply pigments
  • Blend pigments into hybrid colors
  • Stack layers into more complex structures

Simple ingredients gradually become sophisticated manufactured creations.

⸻

Build Automation

Transportation and logistics become the heart of the factory.

Players combine:

  • Belts
  • Underground Belts
  • Balancers
  • Storage
  • Filters

to build continuously operating production lines.

As factories scale, organization becomes just as important as throughput.

⸻

Think With Signals

Advanced progression unlocks the Wire Layer.

This parallel automation system enables:

  • Logic networks
  • Factory control systems
  • Signal generation
  • Automated reactions

Available components include:

  • Levers
  • Comparators
  • Filters
  • Displays
  • Transistors
  • Virtual Processors

Virtual Processors can simulate production entirely through signals before physical production occurs.

User Interface Design

The interface is designed to maximize visible factory space while keeping controls immediately accessible.

Top Bar

Game Menu

Provides access to:

  • Exit current game
  • Upgrade infrastructure
  • Manage markers
  • View statistics
  • Access utility features

Level Indicator

Displays current level.

Selecting it opens the objective screen and delivery requirements.

Goal Item

Displays the current target item.

Selecting it opens a breakdown showing:

  • Item composition
  • Layer arrangement
  • Quadrant layout

This acts as an in-game construction reference.

Layer Toggle

Appears after unlocking Wire Layer.

Allows switching between:

  • Grass Layer (factory construction)
  • Wire Layer (signal construction)

Both occupy the same world space while remaining visually separated.

Marker System

Allows players to:

  • Save locations
  • Display labels while zoomed out
  • Jump to markers directly

Useful for navigating large factories.

⸻

Factory Viewport

The center of the screen displays:

  • Terrain
  • Buildings
  • Conveyors
  • Items
  • Signals
  • Wire networks

This is where all factory interaction occurs.

⸻

Tool Panel

Located in the bottom-left.

Interaction modes include:

Camera Mode

Default navigation mode.

Drag to move the viewport.

Eraser Mode

Drag across tiles to remove structures.

Building Mode

Select buildings to enter placement mode.

⸻

Building Context Menu

Appears in the bottom-right after selecting a building.

Actions include:

  • Placement
  • Rotation
  • Variant switching

Multiple building forms can be selected without leaving construction flow.

⸻

Quick Start Guide

Gesture Guide

Controls adapt depending on the selected tool.

  • Camera Mode → One-finger drag moves the camera
  • Other Modes → Two-finger drag moves the camera
  • Pinch → Zoom
  • Eraser Mode → Drag to continuously remove structures
  • Belt Mode → Drag to continuously place belts
  • Wire Mode → Drag to continuously place wires
  • Overview Mode → One-finger interaction disabled
